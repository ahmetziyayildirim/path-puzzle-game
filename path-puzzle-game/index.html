Path Puzzle Game

Rules

  • Start by placing number 1 anywhere on the grid
  • Place the next number following these movement rules:
    • 3 squares horizontally (left/right) ⬅️➡️
    • 3 squares vertically (up/down) ⬆️⬇️
    • 2 squares diagonally ↗️↘️↙️↖️
  • Game ends when no valid moves are available
  • Your score is the highest number you placed
